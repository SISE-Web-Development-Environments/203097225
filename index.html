    Q&A


    In my opinion, the three main differences between Web 1.0, Web 2.0 and Web 3.0 are:

    Difference Web 1.0 Web 2.0 Web 3.0
    Focus Company Focus: companies publish content that people consume. Community Focus: People publish content that other people can consume, companies build platforms that let people publish content for other people. Individual Focus: People build applications that people can interact with, companies build platforms that let people publish services by leveraging the associations between people or special content.
    Usage mostly Read-Only: It works just as an information portal to the viewers or clients without engaging them or having any sort of interactions. Wildly Read-Write: encourages participation, collaboration, and information sharing. Portable and Personal: executable phrase of WWW with dynamic applications, interactive services and machine interaction languages.
    Accessibility Content is accessible by web forms. Developed APIs to allow self-usage, such as by a software application. Content is accessible by multiple applications, every device is connected to the web, the services can be used everywhere.





    The differences between the use cases "search for "medicines side effects"" on Web 1.0, Web 2.0 and Web 3.0:

    Web 1.0: The user is entering his query to the search engine.
    The results come very fast but there is too much irrelevant or inaccurate information and finding the needed
    information might be like looking for a needle in a haystack.

    Web 2.0: The user is entering his query to the search engine.
    The engine retrieve tags with micro contents and the results are more accurate but part of them might be
    irrelevant or too general because the WWW is not completely mapped at this stage.

    Web 3.0:The user is entering his query to the search engine.
    The results are very accurate and relevant for the user.
